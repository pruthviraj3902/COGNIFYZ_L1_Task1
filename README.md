# 📊 Task 01 - Data Exploration & Preprocessing

## 🎯 Objective
To explore the restaurant dataset, identify missing values, perform preprocessing, and analyze the distribution of aggregate ratings.

## 🛠️ Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Google Colab

## 📌 Steps Performed
1. Loaded the dataset
2. Explored dataset structure and columns
3. Checked for missing values
4. Handled missing values
5. Verified data types
6. Checked dataset shape
7. Analyzed distribution of Aggregate Ratings

## 💡 Key Insights
- Dataset contains multiple restaurant-related features  
- Missing values were identified and handled  
- Aggregate ratings are unevenly distributed  
- Some rating classes appear more frequently than others  
- Data preprocessing improves data quality for analysis
